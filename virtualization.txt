Hardware Virtualization
  - creating a virtual platform of something
  - include virtual computer hardware, virtual storage, virtual computer
  - used hypervisor 

Hardware virtualization types
  - Paravirtualization
  - Full virtualization
  - Hardware-assisted virtualization

Hardware virtualization vs. OS virtualization
  OS virtualization-based systems include Docker, Solaris Containers and Linux-VServer

Hypervisor
  - virtual machine monitor
  - software that creates and runs virtual machines

Benefits of hypervisors
  - speed
  - efficiency
  - flexibility
  - portability
